# 🎓 Student Management System

A comprehensive Student Management System built using **Spring Boot**, **Spring MVC**, **Spring Data JPA**, **JSP**, and **MySQL**. The application enables administrators to efficiently manage student records, courses, attendance, fee details, and reports through an intuitive dashboard.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Add Student
- Update Student
- Delete Student
- View Student Details
- Search Student

### 📚 Course Management
- Add Course
- Edit Course
- Delete Course
- View Course Details

### 📅 Attendance Management
- Mark Daily Attendance
- View Attendance Records
- Attendance Reports

### 💰 Fee Management
- Add Fee Details
- Track Paid and Pending Fees
- Payment Status
- Fee Reports

### 📊 Reports
- Student Reports
- Course Reports
- Attendance Reports
- Fee Reports

### 📈 Dashboard
- Total Students
- Active Students
- Total Courses
- Attendance Overview
- Quick Navigation
- Recent Student Records
